# SAPP

Sistema de gestion de solicitudes, seguimiento y modulos operativos sobre Laravel 12.

## Inicio rapido

Requisitos:
- PHP 8.2+
- Composer
- Node.js 20+
- MySQL/MariaDB

Configuracion:

```bash
composer run setup
```

Entorno de desarrollo:

```bash
composer run dev
```

Pruebas:

```bash
composer run test
```
